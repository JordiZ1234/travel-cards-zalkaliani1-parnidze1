# Travel Cards - Examen CSS MP02
El proyecto se basa en escoger 4 frames de los 12 que el figma ofrece y de esos 4 escogidos copiarlos manualmente con css y HTML
Para la estructura de archivos se ha utilizado el siguiente orden: index.html, card1.html, card2.html, card3.html, card4.html, /images, /css->style.css
Para visualizar la pagina web el procedimiento es el siguiente: Se necesita el addon de live server, posicionar el cursor encima del html, click derecho y abrir con live server
